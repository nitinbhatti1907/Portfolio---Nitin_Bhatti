Browser-based simulator for all 8 CELPIP Speaking tasks with authentic prep and speaking timers, local microphone recording via MediaRecorder API, real-time transcription using the Web Speech API, a live VU meter, image upload for picture-based tasks, and full session persistence — no sign-up, no server, all audio stays in the browser.
